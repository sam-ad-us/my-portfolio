# **App Name**: Portfolio

## Core Features:

- Landing Page: Displays name, role, a brief introduction, and key action buttons (Download CV, GitHub, LinkedIn). Includes a responsive navbar for navigating to different sections.
- Animated Background: Cinematic animated background using Three.js to create a moving 3D grid effect, responding to mouse movement.
- About Section: Displays a profile picture and a description about education, skills, and passions.
- Projects Section: Showcases projects in a card-style layout, with project names, descriptions, tech stack, and links to GitHub/Live Demo.
- Skills Section: Displays programming languages, frameworks, and tools in a grid layout with icons, so you can highlight your strengths.
- Contact Section: Features a contact form for direct messaging and links to social media accounts (GitHub, LinkedIn, Twitter, Instagram).
- CV Enhancement Tool: Uses a generative AI tool to analyze and refine the text in the CV to make it more attractive to employers.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a vibrant and modern look.
- Background color: Dark gray (#222222) to create a dark theme.
- Accent color: Neon pink (#FF69B4) to highlight interactive elements and create visual interest.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused feel.
- Use minimalistic, neon-style icons to represent skills and technologies.
- Employ a clean, responsive grid layout to ensure a consistent experience across devices.
- Implement subtle, smooth animations and hover effects to enhance interactivity.
